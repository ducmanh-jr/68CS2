JavaScript const 1 

                Khi nào sử dụng JavaScript const?
            Luôn khai báo biến const khi bạn biết chắc giá trị của biến đó không nên thay đổi.
            Sử dụng const khi bạn khai báo:
                Một mảng mới
                Một đối tượng mới
                Một chức năng mới
                Một RegExp mới
